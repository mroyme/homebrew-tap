# Madhurjya's Tap

## How do I install these formulae?

`brew install mroyme/tap/<formula>`

Or `brew tap mroyme/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mroyme/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
